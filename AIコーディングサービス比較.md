# ① 全体（ユーザー数・普及率が高い主要サービス）

## ■ 定番・シェア上位

* GitHub Copilot

  * 現在の**業界標準**（最も普及）
  * IDE統合・補完・チャットなど総合力が高い
  * 数百万ユーザー規模

* Cursor

  * AIネイティブIDE（コード全体を理解して編集）
  * 2024〜2025で急成長した注目株

* Google Gemini Code Assist

  * Google製、企業・個人ともに拡大中
  * Copilotに次ぐ利用者層

---

## ■ その他主要プレイヤー

* Amazon CodeWhisperer

  * AWS連携が強い（クラウド開発向け）

* Tabnine

  * ローカル実行・セキュリティ重視（企業向け）

* Claude Code

  * 高度な推論・長文コード処理が強み（近年人気上昇）

* OpenAI Codex

  * API・エージェント系で広く使われる基盤モデル

---

### ▶まとめ（全体）

* **1位：Copilot（安定・王道）**
* **2位：Cursor（新世代IDE）**
* **3位：Gemini / Code系AI（拡大中）**

👉 現場では「複数併用」が普通（約半数が2つ以上使用）

# ② 無料で使える＆無料枠が多いサービス

## ■ 完全無料 or かなり強い無料枠

* Codeium（※現在はWindsurf）

  * **ほぼ完全無料で使える最強候補**
  * 無制限補完＋チャットあり
  * 学生・個人開発に最適

---

* Google Gemini Code Assist

  * 無料で**非常に大きな使用量（例：大量補完）**
  * 学習・個人用途にかなり強い

---

## ■ 無料枠あり（制限付き）

* GitHub Copilot

  * 無料枠あり（例：月2000補完など）
  * 学生は無料になる場合あり

* Cursor

  * 無料プランあり（回数制限あり）

* Amazon CodeWhisperer

  * 個人は無料利用可能

---
