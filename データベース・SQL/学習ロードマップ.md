**① 初級（土台）**

* SELECT / WHERE / ORDER BY / LIMIT
* 集計関数（COUNT, SUM, AVG）
* GROUP BY / HAVING
  👉「1テーブルから正しく欲しいデータを取れる」

**② 初級後半（現場感）**

* INNER / LEFT JOIN（業務系で超重要）
* NULLの扱い
* 日付・文字列関数
  👉「複数テーブルを安全につなげられる」

**③ 中級（差がつく）**

* サブクエリ（WHERE / FROM）
* EXISTS / IN の使い分け
* CASE式
* パフォーマンス基礎（インデックスの考え方、実行計画の見方）
  👉「動くSQL → ちゃんとしたSQL」

**④ 業務系特化トピック**

* マスタ・トラン系テーブル設計の考え方
* 集計帳票SQLの書き方
