## 🇯🇵 国内向け（日本語）

### 🔹 **＠IT（アットマーク・アイティ）**

* 技術トピック別に整理されたITニュース＆技術解説が強み。クラウド、AI、DevOps、セキュリティなど実務に直結したテーマが多いです。

### 🔹 **ITmedia / ITmedia NEWS**

* 国内最大級の総合ITメディア。業界動向、最新テックニュース、製品レビューまで幅広くカバー。エンジニアだけでなくマネジメント層にも人気です。

### 🔹 **TechCrunch Japan**

* 海外のテック／スタートアップ系ニュースを日本語で読める定番サイト。新技術・市場構造・企業動向の把握に有効。

### 🔹 **ZDNet Japan**

* 企業IT、クラウド、セキュリティ、DXなどB2B系ニュースが充実。海外トレンドの日本語翻訳記事も豊富。

### 🔹 **GIGAZINE**

* IT業界全般＋ガジェット最新情報に強い老舗サイト。速報性は高く、トレンド把握にもおすすめ。

### 🔹 **CodeZine / 日経クロステック / マイナビTECH+ / PC Watch / Impress Watch**

* 技術解説やコーディング実例、深掘り系ニュース、業界全体の動向など、**やや専門寄りで実践的な情報が多い**サイト群。

### 🔹 技術共有プラットフォーム（ニュース補完に◎）

* **Qiita / Zenn / gihyo.jp** — エンジニア同士の技術記事、Tips、ツールの最新情報収集に最適。

## 🌍 海外向け（英語）

### 🔹 **TechCrunch（TechCrunch）**

* 世界中のIT／スタートアップ動向、業界分析、先端技術ニュースを網羅。エンジニアだけでなく戦略・ビジネス視点も得られます。

### 🔹 **The Verge**

* テック全般のニュース＆レビューを扱う大手テックメディア。ハード・ソフト両方のトレンド把握に。

### 🔹 **Techmeme**

* テックニュースのリアルタイム集約サイト。重要な話題を一ページで追えるため毎日のチェックに最適。

### 🔹 **TechRadar / CNET / Wired / Ars Technica**

* デジタル／テクノロジー系ニュースを幅広く扱う老舗サイト。製品・セキュリティ・トレンド分析に強い。

### 🔹 **Smashing Magazine / Dev.to / TechRepublic**

* 開発者／エンジニア向け深掘り記事・実装解説・ベストプラクティスが多いサイト。英語ですが実務の勉強に役立ちます。
